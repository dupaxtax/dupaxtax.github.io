Work

Up to this point most of my profession experience with code has been with Java and C++. Both of which have helped me
to understand object oriented programming. Beyond that I've daubbled in Python and Flask to build a website for my
capstone project at school, as well as to create a machine learning model. In my spare time I've been learning what I
can about Scala because I find functional programming intriguing.
